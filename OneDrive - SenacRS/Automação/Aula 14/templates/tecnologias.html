💻 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando diferentes tecnologias para criar uma aplicação web simples e demonstrar o funcionamento de APIs.

🐍 Python

Linguagem de programação utilizada para desenvolver a lógica da aplicação.

🌐 Flask

Framework Python utilizado para criar o servidor web e as rotas da aplicação.

📄 HTML

Linguagem utilizada para estruturar as páginas exibidas no navegador.

📂 JSON

Formato utilizado para armazenar e trocar informações entre sistemas.

🛠️ Visual Studio Code

Editor de código utilizado para desenvolver o projeto.
